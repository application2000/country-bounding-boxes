country-bounding-boxes
======================

Set of the bounding boxes with additional info (name, iso3166, mobile country code) for all countries extracted from http://www.openstreetmap.org/ and http://en.wikipedia.org/wiki/Mobile_country_code

Format of a file:
* `name` - english name of the country
* `iso3166` - ISO 3166-1 alpha-2 code (ref: http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2 )
* `iso3166-numeric` - ISO 3166-1 numeric code (ref: http://en.wikipedia.org/wiki/ISO_3166-1_numeric )
* `mcc` - a list of mobile country codes (ref: http://en.wikipedia.org/wiki/Mobile_country_code)
* `bbox` - a list of bounding boxes (ref: http://geojson.org/geojson-spec.html#bounding-boxes )
